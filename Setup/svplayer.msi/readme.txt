Tools needed:
* WiX 3. (http://wix.sourceforge.net/)

To Compile:
* Start a VS2008 command prompt.
* Change folder to WiX, run build.bat.
* The English version of MSI package will be at bin\Release\en-US; The Chinese versin will be at bin\ReleaseCHS\zh-CN.

Creating Multiple Language MSI
* Start a VS2008 command prompt.
* Change folder to Transform, run build.bat.
* The final MSI file is located at Transform\SvPlayer.msi.
